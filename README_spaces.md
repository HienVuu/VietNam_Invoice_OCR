# Vietnamese OCR with Structured Data Extraction

A web interface for Vietnamese OCR using PaddleOCR and VietOCR, with AI-powered structured data extraction.

## Features

- Upload images of Vietnamese documents (invoices, technical drawings, etc.)
- Automatic text detection and recognition
- Structured data extraction using Google Gemini AI
- View annotated images with bounding boxes
- Download extracted JSON data

## Usage

1. Upload one or more images
2. Click "Process Images"
3. View the annotated image and extracted JSON data

## Requirements

- Google API Key for Gemini integration (set as environment variable GOOGLE_API_KEY)

## Supported Document Types

- Invoices & Receipts
- Technical Specifications
- Engineering Drawings
- Datasheets & Catalogs
- Packing Lists
